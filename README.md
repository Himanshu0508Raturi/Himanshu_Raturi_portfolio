# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, Bootstrap, and JavaScript.

## 📁 Project Structure

```
portfolio/
│
├── index.html          # Main HTML file with all sections
├── README.md           # Project documentation
│
├── css/
│   └── style.css       # Custom CSS styling
│
├── js/
│   └── script.js       # JavaScript functionality
│
└── images/             # Place your images here
    └── (your images)
```

## 🎨 Sections Included

1. **Home** - Hero section with introduction and social links
2. **About Me** - Personal information and skills with progress bars
3. **Academics** - Educational background and qualifications
4. **Projects** - Showcase of projects with descriptions
5. **Certificates** - Professional certifications and achievements
6. **Contact Me** - Contact form and contact information

## 🚀 Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Smooth Scrolling** - Elegant navigation between sections
- **Active Navigation** - Highlights current section in navbar
- **Animated Elements** - Fade-in animations on scroll
- **Progress Bars** - Animated skill progress indicators
- **Back to Top Button** - Easy navigation to top of page
- **Contact Form** - Functional contact form with validation
- **Modern UI** - Clean and professional design
- **Bootstrap Integration** - Responsive grid and components
- **Custom Styling** - Unique color scheme and animations

## 📝 How to Customize

### 1. Personal Information

Open `index.html` and replace the following dummy data:

- **Name**: Search for "John Doe" and replace with your name
- **Title**: Change "Full Stack Developer & Designer" to your title
- **Description**: Update the introduction text in the home section
- **Profile Image**: Replace the placeholder image URL in the home section

### 2. About Section

- Update the personal description paragraphs
- Modify skill names and percentages in the skills section
- Add or remove skills as needed

### 3. Academics Section

- Replace education details (school names, dates, GPAs)
- Update descriptions of your educational achievements
- Add or remove education entries as needed

### 4. Projects Section

- Replace project images (place in `images/` folder)
- Update project titles and descriptions
- Modify technology badges for each project
- Add or remove project cards as needed

### 5. Certificates Section

- Update certificate names and issuing organizations
- Change dates and descriptions
- Update credential links
- Add or remove certificate cards as needed

### 6. Contact Section

- Update location information
- Change email address
- Update phone number
- Modify social media links (GitHub, LinkedIn, Twitter, Instagram)

### 7. Images

Place your images in the `images/` folder and update references in `index.html`:

```html
<!-- Example -->
<img src="images/your-photo.jpg" alt="Your Name">
```

### 8. Colors and Styling

Open `css/style.css` and modify the CSS variables at the top:

```css
:root {
    --primary-color: #0d6efd;    /* Change primary color */
    --secondary-color: #6c757d;  /* Change secondary color */
    --dark-color: #212529;       /* Change dark color */
    --light-color: #f8f9fa;      /* Change light color */
}
```

## 🌐 How to Use

1. **Open Locally**: Simply open `index.html` in your web browser
2. **Edit Content**: Modify the dummy data with your actual information
3. **Add Images**: Place your images in the `images/` folder
4. **Customize Styles**: Edit `css/style.css` to match your preferences
5. **Deploy**: Upload to any web hosting service (GitHub Pages, Netlify, Vercel, etc.)
